# Collaborative Session Part 1: Proto + Desktop Relay

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Extend the protobuf schema and Desktop MQTT relay to support multi-person collaborative sessions — message routing, @Agent detection, noReply context injection, and Agent reply broadcast.

**Architecture:** Extend `teamclaw.proto` with sender fields on ChatRequest and a new CollabControl message. Desktop MQTT relay subscribes to session topics, routes messages to OpenCode (noReply for non-@Agent, normal prompt for @Agent), and broadcasts Agent replies back to the session topic.

**Tech Stack:** Protobuf, Rust (MQTT relay in `mqtt_relay.rs`), OpenCode HTTP API

**Spec:** `docs/superpowers/specs/2026-04-14-collab-session-v1-design.md`

**Dependency:** None — this is the foundation. Parts 2 (iOS login) and 3 (iOS collab UI) depend on this.

```rust
/// Inject a message into OpenCode session history without triggering AI response.
/// Used for collaborative messages that don't @Agent — records context silently.
pub async fn inject_context_no_reply(
    port: u16,
    session_id: &str,
    content: &str,
    sender_name: &str,
) -> Result<(), String> {
    let prefixed = format!("[{}] {}", sender_name, content);

    let client = reqwest::Client::builder()
        .timeout(std::time::Duration::from_secs(10))
        .build()
        .unwrap_or_else(|_| reqwest::Client::new());

    let url = format!(
        "http://127.0.0.1:{}/session/{}/prompt_async",
        port, session_id
    );
    let body = serde_json::json!({
        "parts": [{ "type": "text", "text": prefixed }],
        "noReply": true,
    });

    let resp = client
        .post(&url)
        .json(&body)
        .send()
        .await
        .map_err(|e| format!("inject_context_no_reply failed: {}", e))?;

    if !resp.status().is_success() {
        let status = resp.status();
        let body_text = resp.text().await.unwrap_or_default();
        return Err(format!(
            "inject_context_no_reply HTTP {} - {}",
            status, body_text
        ));
    }

    Ok(())
}
```

**Note:** This assumes OpenCode's `prompt_async` accepts a `noReply` body parameter. If it doesn't, we'll need to verify and potentially use an alternative approach (send message + immediately abort). Test this in Task 5.

```rust
async fn handle_collab_control(
    &self,
    _source_topic: &str,
    ctrl: &proto::CollabControl,
) -> Result<(), String> {
    let session_id = ctrl.session_id.as_deref().unwrap_or("");
    if session_id.is_empty() {
        return Err("CollabControl missing session_id".to_string());
    }

    match proto::CollabControlType::try_from(ctrl.r#type) {
        Ok(proto::CollabControlType::CollabCreate) => {
            eprintln!(
                "[MQTT Relay] Collab CREATE: session={} from={}",
                session_id, ctrl.sender_name
            );

            // Create OpenCode session for this collab
            let opencode_session_id =
                teamclaw_gateway::create_opencode_session(self.opencode_port).await?;

            // Set system prompt for collab session
            let member_names: Vec<&str> = ctrl
                .members
                .iter()
                .map(|m| m.name.as_str())
                .collect();
            let system_prompt = format!(
                "你是一个团队协作助手。当前协作成员：{}。\n当有人 @Agent 时你需要回复。消息格式：[姓名] 内容。",
                member_names.join("、")
            );
            // TODO: Set system prompt on the OpenCode session if API supports it

            // Track the session
            self.collab_sessions
                .lock()
                .await
                .insert(session_id.to_string(), opencode_session_id);

            // Subscribe to the session topic
            let config = self.config.read().await;
            let topic = format!(
                "teamclaw/{}/session/{}",
                config.team_id, session_id
            );
            if let Some(client) = self.client.lock().await.as_ref() {
                client
                    .subscribe(&topic, rumqttc::QoS::AtLeastOnce)
                    .await
                    .map_err(|e| format!("Failed to subscribe to session topic: {}", e))?;
                eprintln!("[MQTT Relay] Subscribed to collab session: {}", topic);
            }
        }
        Ok(proto::CollabControlType::CollabEnd) => {
            eprintln!(
                "[MQTT Relay] Collab END: session={} from={}",
                session_id, ctrl.sender_name
            );

            // Unsubscribe from session topic
            let config = self.config.read().await;
            let topic = format!(
                "teamclaw/{}/session/{}",
                config.team_id, session_id
            );
            if let Some(client) = self.client.lock().await.as_ref() {
                let _ = client.unsubscribe(&topic).await;
            }

            // Remove from tracking
            self.collab_sessions.lock().await.remove(session_id);
        }
        Ok(proto::CollabControlType::CollabLeave) => {
            // Nothing to do on desktop — just a notification
            eprintln!(
                "[MQTT Relay] Collab LEAVE: {} left session {}",
                ctrl.sender_name, session_id
            );
        }
        Err(_) => {
            eprintln!("[MQTT Relay] Unknown CollabControlType: {}", ctrl.r#type);
        }
    }

    Ok(())
}
```

```rust
async fn handle_collab_chat_request(
    &self,
    session_id: &str,
    request: &proto::ChatRequest,
) -> Result<(), String> {
    // Ignore our own Agent broadcasts
    if request.sender_type.as_deref() == Some("agent") {
        return Ok(());
    }

    let sender_name = request.sender_name.as_deref().unwrap_or("Unknown");
    let content = &request.content;

    // Look up OpenCode session ID
    let opencode_session_id = {
        let sessions = self.collab_sessions.lock().await;
        sessions
            .get(session_id)
            .cloned()
            .ok_or_else(|| format!("No OpenCode session for collab {}", session_id))?
    };

    // Check for @Agent mention (case-insensitive)
    let has_agent_mention = content.to_lowercase().contains("@agent");

    if has_agent_mention {
        eprintln!(
            "[MQTT Relay] Collab @Agent from [{}]: {}",
            sender_name,
            &content[..content.len().min(50)]
        );

        // Format message with sender identity
        let formatted = format!("[{}] {}", sender_name, content);
        let prompt_parts = vec![serde_json::json!({
            "type": "text",
            "text": formatted
        })];

        // Send to OpenCode (normal prompt, Agent will respond)
        let port = self.opencode_port;
        let sse_url = format!("http://127.0.0.1:{}/event", port);
        let prompt_url = format!(
            "http://127.0.0.1:{}/session/{}/prompt_async",
            port, opencode_session_id
        );

        let http_client = reqwest::Client::builder()
            .timeout(std::time::Duration::from_secs(900))
            .build()
            .unwrap_or_else(|_| reqwest::Client::new());

        // Connect SSE first
        let sse_response = http_client
            .get(&sse_url)
            .header("Accept", "text/event-stream")
            .send()
            .await
            .map_err(|e| format!("SSE connect failed: {}", e))?;

        // Send prompt
        let body = serde_json::json!({ "parts": prompt_parts });
        http_client
            .post(&prompt_url)
            .json(&body)
            .send()
            .await
            .map_err(|e| format!("Prompt send failed: {}", e))?;

        // Spawn task to collect full response and broadcast
        let relay = self.clone();
        let sid = session_id.to_string();
        let cancel_token = tokio_util::sync::CancellationToken::new();

        tokio::spawn(async move {
            match relay
                .collect_and_broadcast_collab_response(
                    sse_response,
                    &sid,
                    &opencode_session_id,
                    &cancel_token,
                )
                .await
            {
                Ok(()) => {}
                Err(e) => eprintln!("[MQTT Relay] Collab broadcast error: {}", e),
            }
        });
    } else {
        eprintln!(
            "[MQTT Relay] Collab context from [{}]: {}",
            sender_name,
            &content[..content.len().min(50)]
        );

        // No @Agent — inject as silent context
        teamclaw_gateway::inject_context_no_reply(
            self.opencode_port,
            &opencode_session_id,
            content,
            sender_name,
        )
        .await?;
    }

    Ok(())
}
```

- [ ] **Step 3: Implement collect_and_broadcast_collab_response**

This collects the full Agent response from SSE (not streaming to any single device) and broadcasts the complete reply to the session topic:

```rust
async fn collect_and_broadcast_collab_response(
    &self,
    sse_response: reqwest::Response,
    collab_session_id: &str,
    opencode_session_id: &str,
    cancel_token: &tokio_util::sync::CancellationToken,
) -> Result<(), String> {
    use futures_util::StreamExt;

    let mut stream = sse_response.bytes_stream();
    let mut full_content = String::new();
    let mut buffer = String::new();

    loop {
        tokio::select! {
            _ = cancel_token.cancelled() => {
                eprintln!("[MQTT Relay] Collab response cancelled for {}", collab_session_id);
                break;
            }
            chunk = stream.next() => {
                match chunk {
                    Some(Ok(bytes)) => {
                        buffer.push_str(&String::from_utf8_lossy(&bytes));

                        // Process complete SSE lines
                        while let Some(pos) = buffer.find("\n\n") {
                            let event_block = buffer[..pos].to_string();
                            buffer = buffer[pos + 2..].to_string();

                            for line in event_block.lines() {
                                if let Some(data) = line.strip_prefix("data: ") {
                                    if let Ok(parsed) = serde_json::from_str::<serde_json::Value>(data) {
                                        // Extract text content from message.part.delta events
                                        if let Some(content) = parsed
                                            .pointer("/properties/content")
                                            .or(parsed.pointer("/content"))
                                            .and_then(|c| c.as_str())
                                        {
                                            full_content.push_str(content);
                                        }

                                        // Check for completion
                                        let event_type = parsed
                                            .get("type")
                                            .and_then(|t| t.as_str())
                                            .unwrap_or("");
                                        if event_type == "message.done"
                                            || event_type == "message.error"
                                        {
                                            // Broadcast complete reply
                                            if !full_content.is_empty() {
                                                self.broadcast_agent_reply(
                                                    collab_session_id,
                                                    &full_content,
                                                )
                                                .await?;
                                            }
                                            return Ok(());
                                        }
                                    }
                                }
                            }
                        }
                    }
                    Some(Err(e)) => {
                        return Err(format!("SSE stream error: {}", e));
                    }
                    None => {
                        // Stream ended — broadcast whatever we have
                        if !full_content.is_empty() {
                            self.broadcast_agent_reply(collab_session_id, &full_content)
                                .await?;
                        }
                        return Ok(());
                    }
                }
            }
        }
    }

    Ok(())
}

async fn broadcast_agent_reply(
    &self,
    collab_session_id: &str,
    content: &str,
) -> Result<(), String> {
    let config = self.config.read().await;
    let topic = format!(
        "teamclaw/{}/session/{}",
        config.team_id, collab_session_id
    );

    let mut chat_req = proto::ChatRequest::default();
    chat_req.session_id = collab_session_id.to_string();
    chat_req.content = content.to_string();
    chat_req.sender_id = Some("agent".to_string());
    chat_req.sender_name = Some("Agent".to_string());
    chat_req.sender_type = Some("agent".to_string());

    let envelope = build_envelope(proto::mqtt_message::Payload::ChatRequest(chat_req));

    if let Some(client) = self.client.lock().await.as_ref() {
        let payload = proto::encode_message(&envelope);
        client
            .publish(
                &topic,
                rumqttc::QoS::AtLeastOnce,
                false,
                payload,
            )
            .await
            .map_err(|e| format!("Failed to broadcast agent reply: {}", e))?;
        eprintln!(
            "[MQTT Relay] Broadcast Agent reply to {} ({} chars)",
            topic,
            content.len()
        );
    }

    Ok(())
}
```

**Note:** The SSE parsing in `collect_and_broadcast_collab_response` is simplified. The actual SSE event format from OpenCode may differ — check the existing `stream_sse_to_mqtt` method for the exact event parsing logic and adapt accordingly. The key difference is: instead of streaming deltas to a single device, we collect the full response and broadcast once.
